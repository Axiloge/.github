# Axiloge

## About

Axiloge is a small software organization focused on the research, design, and development of computing technologies.

Our work spans multiple areas of software engineering, including software development, scripting and automation, operating system development, artificial intelligence, and other experimental computing projects.

The organization serves as a collaborative environment for building practical solutions, exploring new ideas, and developing technical expertise through real-world projects.
